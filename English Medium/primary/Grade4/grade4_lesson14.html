Lesson 14: Roman Numerals - Let's learn ancient numbers!

For Grade 4 Students (Ages 9-10) - Let's learn to read and write Roman numerals up to 100!

What we will learn today:

In this lesson, you will learn to:

  • Identify the basic **Roman numeral symbols** (I, V, X, L, C).
  • Convert **Roman numerals to standard numbers**.
  • Convert **standard numbers to Roman numerals** (up to 100).
  • Understand the rules for combining Roman numerals.

Roman Numeral Symbols:

Roman numerals use letters to represent numbers.

Key Symbols:

  • I = 1
  • V = 5
  • X = 10
  • L = 50
  • C = 100

Rules:

  • When a smaller value is placed before a larger value, it is subtracted (e.g., IV = 4, IX = 9).
  • When a smaller value is placed after a larger value, it is added (e.g., VI = 6, XI = 11).
  • Only I, X, and C can be repeated (up to three times).
